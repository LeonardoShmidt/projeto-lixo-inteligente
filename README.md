Projeto: Lixo Auto-Reciclável com Arduino
Este repositório apresenta o desenvolvimento de um protótipo de lixo auto‑reciclável, capaz de identificar e separar automaticamente materiais recicláveis (metal, papel e plástico) por meio de sensores e automação com Arduino. O projeto foi desenvolvido no curso de Ciência da Computação da UniFAJ, integrando conceitos de Engenharia de Software, Automação, Programação e Sustentabilidade.

Contexto e motivação
A produção de resíduos sólidos cresce de forma constante e, na prática, muitas pessoas ainda têm dificuldade em separar corretamente o lixo reciclável. Isso prejudica o processo de reciclagem, aumenta o desperdício de materiais reaproveitáveis e gera maior impacto ambiental. O projeto foi pensado como uma solução tecnológica acessível que torna a separação mais simples e incentiva hábitos sustentáveis no dia a dia.
​

Objetivos do projeto
Desenvolver um protótipo funcional capaz de identificar e separar automaticamente metal, papel e plástico.
​

Explorar conceitos de automação, sensores e programação utilizando Arduino.
​

Criar um software para contabilizar a quantidade de itens descartados em cada categoria de material.
​

Propor uma base para integração futura com plataforma web e iniciativas públicas de incentivo à reciclagem.
​

Tecnologias e conceitos utilizados
Arduino para controle do protótipo físico e leitura de sensores de cor.
​

Lógica de programação embarcada para tomada de decisão (direcionamento do resíduo para o compartimento correto).
​

Estrutura física com três compartimentos internos (metal, papel e plástico) e rampa de triagem.
​

Software de contagem de materiais reciclados, com registro de quantidade por tipo de resíduo.
​

Como estudante de Ciência da Computação (bacharelado em andamento) com disciplinas como Engenharia de Software, Banco de Dados, Matemática Discreta e Programação para Dispositivos Móveis e Sistemas Web, utilizo este projeto para consolidar fundamentos de desenvolvimento de sistemas físicos e lógicos, além de boas práticas de organização e documentação.

Funcionamento do protótipo
O material descartado percorre uma rampa inclinada até a área de leitura.
​

Um sensor de cor identifica o tipo de material a partir de cores pré-definidas (ex.: amarelo = metal, vermelho = plástico etc.).
​

O Arduino aciona o mecanismo que direciona o item para o compartimento correto.
​

O software registra a quantidade total de itens encaminhados a cada categoria de reciclável.
​

Resultados esperados
Reconhecimento adequado das cores pelo sensor.
​

Separação eficiente dos materiais simulados e funcionamento estável mesmo após várias utilizações.
​

Geração de dados reais para acompanhamento de reciclagem em ambientes domésticos, educacionais ou públicos.
​

Sensibilização sobre a importância da reciclagem e do uso de tecnologia na solução de problemas ambientais.
